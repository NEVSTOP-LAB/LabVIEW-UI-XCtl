#LabVIEW UI-DisplayTable

用于显示表格数据的LabVIEW 控件。

![测试Main函数界面](http://git.oschina.net/uploads/images/2015/0721/102213_17491dbd_136753.png "测试Main函数界面")

##功能
- 数据以表格的形式显示
- 可修改表头
- 可以切换数据增加的方向，选择增加到第一行或最后一行
- 动态滚动功能
- 动态调整列宽功能
- Filter 功能
- 最大行数限制功能
- 增加Update table 功能，可以作为状态表使用
